# Партии и упаковки

## Партии

Партия (серия) используется для прослеживаемости.

Если включён учёт партий:

- партия может указываться в поступлениях;
- партия может быть обязательной в отгрузках;
- отчёты по остаткам могут строиться в разрезе партий.

## Упаковки

Упаковка используется для учёта товара в таре/упаковочных единицах.

Если включён учёт упаковок:

- можно учитывать содержимое упаковки;
- можно связывать упаковку с партией;
- можно выполнять операции перемещения/отгрузки упаковками.
