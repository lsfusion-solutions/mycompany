---
title: Партии и упаковки
---

## Партии

Партия (серия) используется для прослеживаемости.

Если включён учёт партий:

- партия может указываться в [поступлениях](receipts.md);
- партия может быть обязательной в [отгрузках](shipments.md);
- [отчёты по остаткам](reports-and-ledgers.md) могут строиться в разрезе партий.

## Упаковки

Упаковка используется для учёта товара в таре/упаковочных единицах.

Если включён учёт упаковок:

- можно учитывать содержимое упаковки;
- можно связывать упаковку с партией;
- можно выполнять операции [перемещения](transfers.md)/[отгрузки](shipments.md) упаковками.
