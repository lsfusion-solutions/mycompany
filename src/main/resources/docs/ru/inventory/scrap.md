---
title: Списания
---

Списание используется для фиксации уменьшения остатка по причинам, не связанным с продажей:

- порча;
- потери;
- брак;
- истечение срока;
- внутреннее потребление.

## Типовой сценарий

1. Создайте документ списания.
2. Укажите [место хранения](locations.md).
3. Заполните строки: товар, количество, причина (если ведётся).
4. Завершите документ.

## Связь с другими модулями

Списание может создаваться на основании других документов (например, производственного заказа, если включён соответствующий сценарий).
