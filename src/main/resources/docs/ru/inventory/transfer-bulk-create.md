# Массовое создание перемещений

Инструмент **«Создать перемещения»** предназначен для быстрого создания нескольких документов перемещения из одного источника.

## Где найти

Откройте список **«Склад → Операции → Отгрузки»**.

Действие **«Создать перемещения»** находится в блоке действий списка.

## Когда использовать

Используйте массовое создание, когда нужно подготовить несколько перемещений **из одного места хранения (откуда)** в разные места хранения (куда), а строки товаров вы планируете заполнить позже вручную.

## Условия

Перед запуском убедитесь, что:

1. В списке отгрузок выбран **тип**, который является перемещением.
2. Указано **место хранения (откуда)** — источник.

## Как работает

1. В списке **«Отгрузки»** выберите тип документа — он должен быть типом **«Перемещение»**.
2. Укажите **Место хранения (откуда)** (источник), для которого будут создаваться перемещения.
3. Запустите действие **«Создать перемещения»**.
4. В открывшемся диалоге выберите несколько мест хранения назначения.

## Результат

Система создаст **отдельный документ перемещения для каждого** выбранного места хранения назначения и заполнит в каждом документе:

- **Тип** — выбранный тип «Перемещение»
- **Место хранения (откуда)** — выбранный источник
- **Место хранения (куда)** — соответствующее выбранное назначение

Строки в созданных документах заполняются пользователем.
