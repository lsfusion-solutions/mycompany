---
title: Массовое создание перемещений
---

Инструмент **«Создать перемещения»** предназначен для быстрого создания нескольких документов [перемещения](transfers.md) из одного источника.

## Где найти

Откройте список **«Склад → Операции → [Отгрузки](shipments.md)»**.

Действие **«Создать перемещения»** находится в блоке действий списка.

## Когда использовать

Используйте массовое создание, когда нужно подготовить несколько [перемещений](transfers.md) **из одного [места хранения](locations.md) (откуда)** в разные места хранения (куда), а строки товаров вы планируете заполнить позже вручную.

## Условия

Перед запуском убедитесь, что:

1. В списке [отгрузок](shipments.md) выбран **тип**, который является перемещением.
2. Указано **[место хранения](locations.md) (откуда)** — источник.

## Как работает

1. В списке **«[Отгрузки](shipments.md)»** выберите тип документа — он должен быть типом **«Перемещение»**.
2. Укажите **[Место хранения](locations.md) (откуда)** (источник), для которого будут создаваться перемещения.
3. Запустите действие **«Создать перемещения»**.
4. В открывшемся диалоге выберите несколько мест хранения назначения.

## Результат

Система создаст **отдельный документ перемещения для каждого** выбранного места хранения назначения и заполнит в каждом документе:

- **Тип** — выбранный тип «Перемещение»
- **[Место хранения](locations.md) (откуда)** — выбранный источник
- **[Место хранения](locations.md) (куда)** — соответствующее выбранное назначение

Строки в созданных документах заполняются пользователем.
