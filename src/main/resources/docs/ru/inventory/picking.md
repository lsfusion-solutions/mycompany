---
title: Задания на сборку
---

Задания на сборку используются для организации отбора товара из [места хранения](locations.md) при [отгрузке](shipments.md).

## Общая логика

1. По [отгрузке](shipments.md) формируются задания на сборку.
2. Кладовщик выполняет сборку (часто через мобильный интерфейс):
   - сканирует ячейку/товар (если включено);
   - вводит фактическое отобранное количество.
3. Выполненные задания обновляют данные в [отгрузке](shipments.md).

## Мобильный интерфейс

В мобильной форме обычно доступны:

- список заданий;
- сканирование штрихкодов;
- подтверждение отбора.
