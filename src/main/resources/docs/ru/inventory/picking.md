# Задания на сборку

Задания на сборку используются для организации отбора товара из места хранения при отгрузке.

## Общая логика

1. По отгрузке формируются задания на сборку.
2. Кладовщик выполняет сборку (часто через мобильный интерфейс):
   - сканирует ячейку/товар (если включено);
   - вводит фактическое отобранное количество.
3. Выполненные задания обновляют данные в отгрузке.

## Мобильный интерфейс

В мобильной форме обычно доступны:

- список заданий;
- сканирование штрихкодов;
- подтверждение отбора.
