# Смены

Смена — это период работы кассы между открытием и закрытием. Продажи и возвраты в POS выполняются в рамках открытой смены.

## Где находится

**«Розница» → «Операции» → «Смены»**.

## Как открыть смену

1. Откройте список смен.
2. Выберите кассу.
3. Выполните действие **«Открыть смену»**.

Система автоматически фиксирует дату и время открытия.

### Ограничения

- Нельзя открыть смену, если по выбранной кассе уже есть открытая смена.

## Как закрыть смену

1. Откройте смену, которую нужно закрыть.
2. Выполните действие **«Закрыть смену»** и подтвердите.

Система фиксирует дату и время закрытия.

### Ограничения

- Смена не может быть закрыта раньше, чем была открыта.

## Контроль операций по смене

В списке смен обычно доступна фильтрация по дате. В POS, как правило, можно просматривать список чеков:

- по текущей смене;
- по кассе.
