title: Настройки розницы
---

Страница описывает базовые настройки, влияющие на работу **[кассы](pos.md)** и **[POS](pos.md)**.

## Где находится

Настройки обычно находятся в разделе **«Розница» → «Настройка»**.

В большинстве конфигураций основные справочники (кассы, способы оплаты, дисконтные карты) доступны прямо из этого раздела.

## Кассы

Касса — это рабочее место, с которого оформляются продажи и возвраты.

Где искать: обычно **«Розница» → «Настройка» → «Кассы»**.

Обычно настраиваются:

- **наименование** и **код** кассы;
- **организация**;
- привязка кассы к **компьютеру** (чтобы на конкретном компьютере предлагалась «своя» касса).

## Смены

Для **[смен](sessions.md)** обычно используется нумерация. В настройках может быть доступен параметр нумератора смен.

## Способы оплаты

В настройках задаётся список способов оплаты (см. также: **[Оплата в рознице](payments.md)**). Для каждого способа обычно указываются:

Где искать: обычно **«Розница» → «Настройка» → «Способы оплаты»**.

- наименование и код;
- признак **«наличные»** (используется для расчёта сдачи);
- сопоставление с финансовыми типами оплат (если используется финансовый контур).

## Дисконтные карты

Для **[дисконтных карт](discount-cards.md)** может использоваться автоматическая нумерация. В настройках может быть доступен параметр нумератора дисконтных карт.

Где искать: обычно **«Розница» → «Настройка» → «Дисконтные карты»**.

## Маркированные товары

Если в вашей конфигурации используется работа с **[маркированными товарами](marking.md)**:

- по категориям может задаваться обязательность сканирования;
- может настраиваться режим проверки кодов (онлайн/оффлайн);
- для оффлайн-проверки может потребоваться указать локальный модуль для кассы.
