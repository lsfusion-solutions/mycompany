# Отчётность

Страница описывает типовые варианты отчетности по проектам, задачам и трудозатратам. Конкретный набор отчетов и показателей зависит от настроек.

## Для чего нужна отчетность по проектам

Отчетность помогает:

- контролировать прогресс по проектам и задачам;
- видеть отклонения по срокам;
- анализировать трудозатраты по проектам, задачам и участникам;
- находить «узкие места» (перегрузка сотрудников, блокирующие зависимости, просрочки).

## Типовые отчёты и разрезы

#### По проектам

Обычно интересуют:

- список активных проектов;
- проекты с истекающими сроками;
- проекты без задач или без команды (если ведется команда).

#### По задачам

Обычно интересуют:

- задачи по статусам (в работе, ожидают, завершены);
- просроченные задачи;
- задачи без исполнителя;
- задачи с блокировками (по зависимостям).

#### По трудозатратам

Обычно интересуют:

- трудозатраты по проектам за период;
- трудозатраты по задачам за период;
- трудозатраты по сотрудникам за период;
- сравнение «план/факт» (если планирование трудозатрат используется в организации).

## Рекомендованный регламент контроля

#### Ежедневно

- команда обновляет статусы задач;
- исполнители вносят отметки времени.

#### Еженедельно

- менеджер проекта проверяет просрочки и приоритеты;
- уточняет зависимости и блокировки;
- пересматривает сроки ближайших задач.

#### По завершению проекта

- перевод проекта в закрытый статус;
- фиксация итогов и ключевых решений в комментариях/описании;
- подготовка итогового отчета по трудозатратам.

## Частые вопросы

#### Почему отчет по трудозатратам «не сходится»

Проверьте:

- внесли ли все участники отметки времени за период;
- не было ли ввода в неправильный проект или задачу;
- не ограничен ли ввод в закрытом периоде;
- корректно ли выбран период отчета.
