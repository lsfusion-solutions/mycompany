---
title: Настройка
---

Раздел «Настройка» содержит параметры и справочники, которые определяют работу «Кадров»: статусы и правила подбора, параметры учёта времени и параметры расчёта/выплаты зарплаты.

Набор доступных настроек зависит от конфигурации вашей организации и прав пользователя.

![Форма «Настройки» модуля «Кадры»](images/settings.png)

## Подбор персонала

Обычно настраиваются:

- **причины отказа** — у каждой может быть шаблон письма, отправляемого при отказе по этой причине;
- **ярлыки анкет** для классификации анкет;
- **шаблоны писем** для коммуникации с кандидатом;
- почтовый ящик для автоматической обработки входящих сообщений по анкетам (если используется).

Набор статусов анкеты фиксирован («Новый», «Собеседование», «Нанят», «Отказано»); на вкладке **«Статус анкеты»** каждому статусу можно задать цвет и признак «только просмотр».

## Регистрация времени

Обычно настраиваются:

- возможность регистрации времени без геолокации;
- параметры киоска регистрации времени (в том числе использование фотографии);
- шаблоны печати для бейджей (если используются бейджи).

## Расчёт и выплата зарплаты

Обычно настраиваются:

- **типы расчётных листов** (например, Стандартный) и их нумерация;
- **виды расчёта** — категории начислений и удержаний, используемые в строках расчётного листа;
- тип платежа, который используется при регистрации выплат по расчётным листам (если организация фиксирует выплаты в системе).
