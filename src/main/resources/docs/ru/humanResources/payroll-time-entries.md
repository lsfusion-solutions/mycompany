---
title: Выплата по отмеченному времени (раздел «Проекты»)
---

В некоторых организациях часть начислений может рассчитываться на основании **отметок времени** из раздела **«Проекты»** (учёт трудозатрат по проектам и задачам).

Как правило, такие начисления отражаются в расчётном листе отдельными строками (например, по каждому проекту), а исходные записи можно посмотреть во вкладке **«Отметки времени»**.

## Что учитывается в расчёте

При формировании расчётного листа за период система берёт **все отметки времени сотрудника** за этот период, у которых выбран **проект**, и суммирует часы **по каждому проекту**.

Если отметка времени создана **без проекта**, она **не участвует** в расчёте зарплаты по отмеченному времени.

## Как считается сумма

Для каждого проекта рассчитывается начисление:

- **часы** — сумма часов по отметкам времени сотрудника за период (по данному проекту);
- **сумма** — `часы × «Зарплата в час»`.

Сумма начисления округляется до 2 знаков.

## Откуда берётся «Зарплата в час»

Ставка **«Зарплата в час»** может задаваться в зависимости от используемой в вашей организации модели:

- на уровне сотрудника;
- на уровне назначения сотрудника на проект;
- на уровне команды (если применяется).

Для конкретной отметки времени используется ставка, которая соответствует сотруднику и проекту **на дату отметки**.

## Где проверить исходные данные

Если сумма по отмеченному времени получилась неверной или не сформировалась:

1. Проверьте, что отметки времени введены за нужный период и у них выбран проект.
2. Проверьте ставку **«Зарплата в час»** для сотрудника/проекта.
3. Откройте расчётный лист и просмотрите вкладку **«Отметки времени»** — в ней видны исходные записи за период.

Подробности по вводу отметок времени см. в документации раздела «Проекты»: [Отметки времени](../projectManagement/time-entries.md).
