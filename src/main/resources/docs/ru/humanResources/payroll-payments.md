---
title: Выплата зарплаты и контроль выплат
---

В некоторых организациях выплаты фиксируются в системе: по расчётному листу можно зарегистрировать платёж и контролировать, сколько уже выплачено и сколько осталось.

Если в вашем интерфейсе нет действий для регистрации выплат, значит функция может быть **не включена** настройками или недоступна по правам.

## Зарегистрировать выплату по расчётному листу

Рекомендуемый порядок работы:

1. Откройте расчётный лист сотрудника.
2. Проверьте итог **«К выдаче»**.
3. Выполните действие **«Зарегистрировать платёж»**.
4. Проверьте сумму выплаты.
   - Если выплата частичная, уменьшите сумму.
   - Сумму можно скорректировать в пределах доступного остатка.
5. Сохраните платёж.

После регистрации платёж учитывается в контроле «начислено / выплачено / осталось».

## Контроль «начислено / выплачено / осталось»

При включённой регистрации выплат система может показывать сводные показатели за период:

- **начислено «к выдаче»** — сумма итогов «К выдаче» по расчётным листам за период;
- **выплачено** — сумма зарегистрированных выплат за период;
- **осталось к выплате** — разница между начислено и выплачено.

#### На что обратить внимание

- Если выплата по расчётному листу зарегистрирована частично, «осталось к выплате» уменьшится не до нуля.
- Если по сотруднику есть несколько расчётных листов за период, контроль обычно строится по сумме всех документов за выбранный диапазон дат.
