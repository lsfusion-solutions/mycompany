title: Настройки и справочники
---

## Где находится

Откройте **«Расчёты» → «Настройка» → «Настройки»**.

## Что обычно настраивается

- типы документов ([поступления](bills.md), [реализации](invoices.md), [платежи](payments.md));
- нумераторы;
- банковские счета и кассы;
- условия оплаты (см. [Задолженность и календарь платежей](debt-and-calendar.md));
- [налоги](taxes.md);
- параметры распознавания поступлений по файлу с помощью OpenAI (если используется);
- шаблоны печати (см. [Печать и отчётность](reports-and-printing.md));
- правила распределения платежей (если включены).

## Банки и счета

В справочниках ведутся:

- банки;
- банковские счета;
- кассы;
- аналитические счета (если используются для разнесения платежей).

## Условия оплаты

Условия оплаты используются для:

- расчёта даты плановой оплаты;
- формирования [календаря платежей](debt-and-calendar.md);
- контроля просрочки.

## Импорт поступлений по файлу

Если в конфигурации используется распознавание поступлений по файлу, заранее подготовьте две группы настроек:

- в общих параметрах интеграции OpenAI заполните ключ доступа. **Модель по умолчанию** можно указать дополнительно; если она не задана, используется стандартная модель сервиса;
- в карточке **типа поступления** заполните **промпт** для распознавания. При первом заполнении удобно подставить текст **по умолчанию**, а затем при необходимости уточнить его под ваши документы.

Также перед запуском распознавания проверьте справочники:

- поставщики и [номенклатура](../masterdata/items.md) должны быть заведены заранее и быть активными;
- [валюты](../masterdata/currencies.md) и [налоги](taxes.md) должны уже существовать в системе.

Кнопка импорта появляется в карточке поступления только для тех типов, у которых заполнен промпт.
