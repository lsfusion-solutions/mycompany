# Настройки и справочники

## Где находится

Откройте **«Расчёты» → «Настройка» → «Настройки»**.

## Что обычно настраивается

- типы документов (поступления, реализации, платежи);
- нумераторы;
- банковские счета и кассы;
- условия оплаты;
- налоги;
- шаблоны печати;
- правила распределения платежей (если включены).

## Банки и счета

В справочниках ведутся:

- банки;
- банковские счета;
- кассы;
- аналитические счета (если используются для разнесения платежей).

## Условия оплаты

Условия оплаты используются для:

- расчёта даты плановой оплаты;
- формирования календаря платежей;
- контроля просрочки.
