# Распределение на себестоимость

Распределение на себестоимость позволяет включить стоимость услуг (например, фрахт, таможенные пошлины, страхование) в себестоимость товаров или производственных заказов.

## Настройка

### База распределения для услуг
Для каждой услуги, которую необходимо распределять, можно указать **Базу распределения** по умолчанию. Она определяет, как стоимость услуги будет пропорционально разделена между целями.

1. Перейдите в карточку **Товара** (для услуги).
2. В поле **База распределения** выберите один из вариантов:
    - **Сумма** — распределение пропорционально сумме строк-целей без налогов.
    - **Собебестоимость** — распределение пропорционально себестоимости товаров.
    - **Цена продажи** — распределение пропорционально цене продажи товаров.
    - **Вес** — распределение пропорционально весу товаров.
    - **Объем** — распределение пропорционально объему товаров.
    - **Количество** — распределение пропорционально количеству товаров или производимых изделий.

## Распределение на себестоимость в Поступлении

При добавлении строки с услугой в **[Поступление](bills.md)**, вы можете распределить её стоимость на другие документы.

1. Откройте **Поступление** и выберите строку с услугой в списке товаров.
2. На вкладке **Распределение на себестоимость** (в нижней части формы) вы увидите настройки распределения для этой строки.
3. **База распределения** заполняется автоматически из карточки услуги, но её можно изменить для конкретной строки поступления.
4. Используйте подвкладки для выбора целей распределения:

### Распределение на другие Поступления
На вкладке **Поступления**:
- Вы видите список поступлений.
- Отметьте флажком **Распределить** те поступления, которые вы хотите включить в распределение.
- В списке **Строки поступления** ниже можно уточнить, на какие именно строки с товарами должна лечь стоимость.
- В колонке **Распределено** отображается сумма, назначенная на каждую строку на основе выбранной **Базы распределения**.

### Распределение на Производственные заказы
На вкладке **Производственные заказы**:
- Вы видите список открытых производственных заказов.
- Отметьте флажком **Распределить** заказы, которые хотите включить.
- Стоимость будет распределена между выбранными заказами.

### Выполнение распределения
- Нажмите кнопку **Распределить** на панели инструментов строк услуг, чтобы автоматически рассчитать и заполнить колонку **Распределено** для всех выбранных целей.
- Система следит за тем, чтобы общая распределенная сумма не превышала сумму услуги без налогов.

## Влияние на себестоимость
Распределенные затраты включаются в общую себестоимость целевых товаров или производственных заказов, влияя на итоговую оценку запасов или стоимость выпуска.
