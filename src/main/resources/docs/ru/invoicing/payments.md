# Платежи

Платежи в разделе **«Расчёты»** делятся на два вида:

- **входящие** — поступление денег от контрагента;
- **исходящие** — списание денег контрагенту.

Подробные инструкции вынесены в отдельные статьи:

- [Входящие платежи](incoming-payments.md)
- [Исходящие платежи](outgoing-payments.md)

## Печать

Если в вашей конфигурации подключены печатные формы, платежи можно распечатывать.

Подробнее: [Печать и отчётность](reports-and-printing.md).
