## Где находится

Откройте **«Продажи» → «Настройка» → «Настройки»**.

![Настройки модуля «Продажи»](images/settings.png)

## Что обычно настраивается

### Типы заказа

Для каждого типа заказа можно задать:

- **нумератор** — формат и счётчик номеров;
- **валюту по умолчанию** и признак «цена включает налоги»;
- **вид цены** — вид цены для подстановки цен в строки заказа;
- **тип отгрузки** (если включён модуль Склад) — какой документ отгрузки создаётся при подтверждении заказа;
- **тип заказа на производство** — позволяет создавать заказы на производство из заказа покупателя;
- **«Автоматически создавать производственный заказ»** (отображается, когда задан тип заказа на производство) — автоматически создаёт заказ на производство при подтверждении заказа покупателя;
- **тип реализации** и **политику выставления реализаций** — тип реализации для действия «Создать реализацию» и то, берутся ли строки реализации из заказанного или из отгруженного количества;
- **шаблон письма** — шаблон по умолчанию, тему, тело и адрес копии для действия «Отправить» (см. [статус «Отправлен»](workflow-and-statuses.md));
- **«Запретить закрытие заказов с действующими отгрузками»**, **«Запретить закрытие заказов, которые не полностью отгружены»** и **«Запретить закрытие заказов, которые не полностью оплачены»** — ограничения перехода в статус «Закрыт».

### Глобальные настройки модуля

На форме настроек модуля доступны общие переключатели:

- **«Не рассчитывать скидки автоматически в заказе»** — отключает автоматический пересчёт скидок при изменении строк (полезно при ручном управлении скидками);
- **«Не рассчитывать скидки автоматически в реализации»** — то же для документов реализации.

См. также: [Скидки](discounts.md#автоматический-пересчёт-скидок).

### Статьи продаж

Вкладка **«Статьи продаж»** формы настроек содержит список статей продаж. Они используются на вкладке **«Продажи по статьям»** [отчета по продажам](reports.md).

### Прайс-листы

- **Типы прайс-листа** — категории, по которым строится список (например, «Стандартный», «Акционный»);
- **Виды цены** — используются в строках прайс-листов и заказов для определения цены; ведутся в отдельной форме: **«Продажи» → «Настройка» → «Виды цены»**;
- **Шаблоны печати** — для печати прайс-листов.

### Что ещё

- параметры печати (шаблоны для заказов и сопроводительных документов);
- доступность отдельных действий в зависимости от статусов.

Рекомендация: сначала настройте типы заказов и нумераторы, затем виды цены и прайс-листы, и в последнюю очередь — скидки (они опираются на виды цены и категории).
