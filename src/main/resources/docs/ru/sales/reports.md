Раздел отчетности используется для анализа:

- объёма продаж;
- выполнения заказов;
- отгрузок и реализаций;
- скидок;
- дебиторской задолженности (если включено).

## Отчет по ценам

**Отчет по ценам** позволяет анализировать цены на продукцию на определенные моменты времени, сравнивать их между двумя датами и просматривать остатки на выбранных складах.

### Основные возможности

- **Анализ цен**: Просмотр цен для всех или выбранных типов цен на определенную дату и время.
- **Сравнение цен**: Сравнение цен между двумя датами. Отчет показывает цену на обе даты и процентную разницу.
- **Визуальные индикаторы**: Повышение цены выделяется **зеленым** цветом, а понижение — **красным**.
- **Интеграция с запасами**: Просмотр текущих остатков (включая вложенные склады) для каждой позиции на выбранные даты и склады.
- **Фильтрация**:
    - По **Типу цены**: Сосредоточьтесь на конкретном типе цены или просмотрите все.
    - По **Складу**: Фильтруйте товары по их наличию на конкретном складе.
    
### Как использовать

1. Перейдите в раздел **Продажи** -> **Отчетность** -> **Отчет по ценам**.
2. Выберите целевую **Дату**.
3. (Опционально) Выберите дату в поле **Сравнить**, чтобы увидеть изменения цен и запасов.
4. (Опционально) Выберите **Тип цены**, чтобы отфильтровать отображаемые цены.
5. (Опционально) Выберите **Склад**, чтобы увидеть уровни запасов и отфильтровать товары, которые есть в наличии на этом складе.

Отчет отображает список товаров с их атрибутами, штрих-кодами, единицами измерения, уровнями запасов и ценами на выбранные даты.

## Рекомендации

1. Используйте интервал дат.
2. Группируйте по контрагентам и номенклатуре.
3. Сверяйте суммы заказов и суммы реализаций, чтобы контролировать выполнение.
