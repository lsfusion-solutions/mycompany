# Реализации по заказам

Реализация фиксирует продажу в учётном контуре: выручку, налоги и итоговые суммы.

## Где находится

Обычно доступно в разделе **«Продажи» → «Операции»** (либо в разделе «Учёт» — зависит от конфигурации).

## Связь с заказом

Реализация может создаваться на основании заказа покупателя и/или отгрузки.

При создании обычно переносятся:

- контрагент;
- место хранения/адрес;
- строки и количества (по заказу или по факту отгрузки);
- цены, скидки и налоги.

## Типовой сценарий

1. Убедитесь, что заказ подтверждён.
2. Создайте реализацию по заказу (или по отгрузке, если так настроено).
3. Проверьте суммы и налоги.
4. Проведите документ.
