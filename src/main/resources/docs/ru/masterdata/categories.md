title: Категории
---

Справочник **«Категории»** используется для группировки номенклатуры. Категории могут быть иерархическими (дерево): у категории может быть родительская категория.

## Список и дерево

Обычно доступны два представления:

- **Список** — плоский перечень категорий;
- **Дерево** — иерархия категорий.

![Список категорий](images/categories.png)

## Карточка категории

Типовые реквизиты:

- **Имя** (наименование);
- **Код** (может формироваться автоматически);
- **Родитель** — обязательное поле: у каждой категории, кроме корневой, есть родительская категория;
- **Неактивно** — признак архивной записи.

В карточке есть поле **«Префикс»** наименования — оно подставляется в начало **полного имени** всех позиций этой категории. На вкладке **«Значения по умолчанию»** задаётся **единица измерения** по умолчанию: она подставляется (с учётом ближайшей родительской категории, где она задана) при установке категории у позиции с пустой единицей измерения; уже заполненную единицу она не меняет, и изменение значения по умолчанию не обновляет существующие позиции.

## Создание подкатегории

В представлении **«Дерево»** выберите родительскую категорию и нажмите кнопку **«Категория»** на панели инструментов — новая категория будет создана как дочерняя к выбранной. В представлении **«Список»** создайте новую запись и укажите родителя вручную.

## Ограничения

Если категория используется в других категориях (как родитель) или в номенклатуре, её удаление может быть запрещено. В таком случае используйте архивирование.
