---
title: Валюты и курсы валют
---

Справочники **«Валюты»** и **«Курсы валют»** используются для расчётов в разных валютах и корректного пересчёта сумм по датам.

## Валюты

### Карточка валюты

Типовые реквизиты:

- **Код** (например, трёхсимвольный);
- **Наименование**;
- **По умолчанию** — валюта, которая подставляется автоматически при создании новых данных, если это предусмотрено.

Рекомендуется:

- иметь одну валюту, отмеченную как **валюта по умолчанию**;
- не создавать дубликаты валют с одинаковым кодом.

## Курсы валют

Курс хранится как набор строк с датами и значениями.

### Карточка курса валют

В карточке курса обычно указываются:

- **наименование**;
- **валюта**, для которой ведётся курс;
- признак **по умолчанию** (для выбранной валюты).

Ниже — таблица строк курса:

- **валюта** (валюта, из которой выполняется пересчёт);
- **дата**;
- **курс**.

### Как используется дата

При пересчёте по документу/операции выбирается значение курса на **последнюю дату, не превышающую дату документа**. Поэтому важно регулярно добавлять новые строки курсов при изменении значений.

### Практические рекомендации

- Вносите курсы с правильными датами действия.
- Если курс изменился, добавляйте новую строку с новой датой (не редактируйте «исторические» значения без необходимости).
