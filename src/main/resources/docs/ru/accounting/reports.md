## Где найти

Откройте **Бухгалтерия -> Отчетность -> Главная книга**.

## Главная книга

Главная книга - основной рабочий экран бухгалтерии. В шапке можно задать:

- **Интервал дат**
- **Компанию**
- **Дату запрета изменений**

Доступные действия:

- **Сформировать проводки** - массово создает отсутствующие проводки за выбранные компанию и период;
- **Удалить проводки** - массово удаляет сформированные из исходных документов проводки за выбранные компанию и период.

Сама форма показывает:

- дерево счетов;
- **Начальный остаток**, **Дебет**, **Кредит** и **Конечный остаток** по каждому счету;
- фильтр **Ненулевые**, включенный по умолчанию;
- строки проводок по выбранному счету.

Из области строк можно:

- открыть исходный документ;
- открыть саму проводку.

Практический сценарий:

1. Укажите компанию и период.
2. При необходимости сформируйте отсутствующие проводки.
3. Проверьте остатки по счетам.
4. Расшифруйте суммы до строк проводок и исходных документов.
5. После проверки и закрытия периода установите дату запрета изменений.

## Баланс

Отчет **«Баланс»** печатается из Главной книги.

Он показывает остатки на конец даты по:

- счетам активов;
- счетам обязательств;
- счетам капитала.

## Отчет о прибылях и убытках

**Отчет о прибылях и убытках** также печатается из Главной книги за выбранный интервал дат.

Он показывает:

- счета доходов;
- счета расходов;
- итоги по **Доходам**, **Расходам** и **Чистой прибыли**.

## Отчет о движении денежных средств

**Отчет о движении денежных средств** печатается из Главной книги за выбранный интервал дат.

Он показывает:

- статьи движения денежных средств по видам деятельности;
- рассчитанную денежную сумму по каждой статье;
- входящий остаток денежных средств;
- исходящий остаток денежных средств.

Отчет зависит от корректной настройки:

- флага **Денежный эквивалент** у счетов Главной книги;
- назначений **Статей ДДС (Дебет / Кредит)** у счетов Главной книги.
