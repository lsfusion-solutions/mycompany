## Где найти

Откройте:

- **Бухгалтерия -> Операции -> Основные средства**
- **Бухгалтерия -> Операции -> Амортизация ОС**

## Карточка основного средства

Карточка ОС содержит:

- **Номер**
- **Номенклатуру**
- **Компанию**
- **Тип**
- **Дату приобретения**
- **Дату начала амортизации**
- **Срок полезного использования, месяцев**
- **Стоимость приобретения**
- **Ликвидационную стоимость**
- **Амортизируемую стоимость**
- **Ежемесячную амортизацию**
- **Дату окончания амортизации**
- **Примечание**

Значения по умолчанию, которые подставляет система:

- **Компания** берется из компании по умолчанию;
- **Дата начала амортизации** равна дате приобретения;
- **Срок полезного использования** подставляется из выбранного типа ОС;
- **Номер** генерируется по нумератору типа ОС.

## Вкладки карточки ОС

В карточке ОС есть вкладки:

- **Амортизация** - плановые строки амортизации;
- **Проводки** - проводки по приобретению и амортизации, связанные с объектом;
- **История** - журнал изменений компании и типа;
- **Файлы**
- **Комментарии**

## Расчет амортизации

Используйте действие **«Рассчитать амортизацию»** в карточке ОС, чтобы построить ежемесячные строки амортизации.

Расчет:

- создает по одной строке на каждый месяц срока полезного использования;
- пропорционально рассчитывает первый месяц от даты начала амортизации до конца месяца;
- корректирует последний месяц так, чтобы общая сумма совпала с полной амортизируемой стоимостью.

В списке основных средств также показывается текущая **Балансовая стоимость**.

## Проводки по основным средствам

Используйте **Сформировать** или **Переформировать** на вкладке **«Проводки»** в карточке ОС.

Модуль создает:

- одну проводку по приобретению на дату и сумму приобретения;
- проводки по амортизации для строк амортизации до текущей даты.

При формировании используются журнал и счета, заданные в типе ОС:

- **Счет основных средств**
- **Счет поступления**
- **Счет амортизации**
- **Счет расходов**

Если задана дата запрета изменений, формирование пропускает дату приобретения или даты амортизации, которые попадают до этой даты.

## Список амортизации ОС

Список **«Амортизация ОС»** показывает:

- номер ОС;
- тип ОС;
- наименование ОС;
- компанию;
- дату амортизации;
- сумму амортизации;
- номер связанной проводки;
- признак проведения связанной проводки.

## Создание основного средства из поступления

Если у типа поступления указан тип ОС, объекты основных средств можно создавать прямо из строк поступления. См. [Проводки из исходных документов](source-documents.md).
