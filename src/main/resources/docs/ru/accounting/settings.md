## Где найти

Откройте:

- **Бухгалтерия -> Настройка -> Настройки**
- **Бухгалтерия -> Настройка -> План счетов**
- **Бухгалтерия -> Настройка -> Статьи движения денежных средств**

## Настройки

Форма **«Настройки»** содержит:

- **Только чтение для проведенных проводок** - если включено, проведенные проводки становятся доступными только для просмотра, пока не будет снято проведение;
- нумератор, используемый для ручных [проводок](journal-entries.md);
- вкладки **Журналы** и **Типы основных средств**.

**Дата запрета изменений** задается в шапке [Главной книги](reports.md). Она используется как жесткая граница для редактирования проводок.

## План счетов

План счетов представляет собой дерево. У каждого счета есть:

- **Код**
- **Наименование**
- **Тип**
- **Родитель**
- **Денежный эквивалент**
- **Статья ДДС (Дебет / Кредит)**

Важные правила:

- тип счета должен быть задан у самого счета либо наследоваться от одного из родителей;
- флаг **Денежный эквивалент** используется в [отчете о движении денежных средств](reports.md);
- назначения статей ДДС наследуются от родительских счетов, если они не заданы непосредственно у дочернего счета.

Если база инициализируется без счетов, модуль загружает типовой план счетов с основными группами:

- Активы
- Обязательства
- Капитал
- Доходы
- Расходы

и типовыми субсчетами, например: Касса, Банк, Дебиторская задолженность, Кредиторская задолженность, Основные средства, Накопленная амортизация, Задолженность по заработной плате, Налоги к уплате.

## Журналы

Журналы используются в типах документов и ручных проводках. У журнала есть:

- **Код**
- **Наименование**
- **Нумератор**

Если база инициализируется без журналов, модуль создает типовые журналы:

- Банк
- Касса
- Продажи
- Закупки
- Прочее

## Статьи движения денежных средств

Статьи ДДС классифицируют движения для отчета о движении денежных средств. У каждой статьи есть:

- **Наименование**
- **Вид деятельности**: Операционная, Инвестиционная или Финансовая

Затем эти статьи назначаются счетам Главной книги как классификация по дебету и/или кредиту.

## Типы основных средств

Типы основных средств используются при создании объектов ОС и начислении амортизации. Тип определяет:

- **Наименование**
- **Нумератор**
- **Журнал**
- **Счет основных средств**
- **Счет поступления**
- **Счет амортизации**
- **Счет расходов**
- **Срок полезного использования, месяцев**

Если база пуста, модуль также создает тип по умолчанию с именем **Asset**.

## Что настраивается вне этого раздела

Правила формирования проводок для поступлений, реализаций, платежей, приемок, отгрузок, инвентаризаций, списаний и расчетных листов настраиваются в карточках соответствующих типов документов в других модулях. См. [Проводки из исходных документов](source-documents.md).
