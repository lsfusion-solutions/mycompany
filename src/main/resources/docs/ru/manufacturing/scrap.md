---
title: Брак и списания
---

В рамках [производственного заказа](orders.md) может быть оформлен документ списания/брака, связанный с заказом.

## Когда использовать

Используйте списание/брак, если требуется списать материалы или продукцию по причине брака и важно сохранить связь с [производственным заказом](orders.md).

## Как создать списание из производственного заказа

В карточке [производственного заказа](orders.md) может быть доступно действие создания списания.

Особенности:

- действие доступно, если для типа производственного заказа задан тип списания;
- в создаваемом документе автоматически подставляется [производственный заказ](orders.md);
- место хранения берется из места хранения материалов заказа.

## Где посмотреть связанные списания

В карточке [производственного заказа](orders.md) отображается блок связанных документов списания. Из него можно открыть документ для просмотра и редактирования.

## Рекомендации

- оформляйте списание до завершения заказа, чтобы фактические данные были согласованы;
- проверяйте, что выбрано корректное место хранения материалов.
