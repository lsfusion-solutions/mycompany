## Где находится

Основные отчёты находятся в разделе **«Закупка» → «Отчётность»**.

## Отчёт по заказам поставщикам

Отчёт по заказам позволяет анализировать заказы в разрезе:

- [поставщиков](../masterdata/partners.md);
- статусов;
- дат;
- [номенклатуры](../masterdata/items.md) и категорий (если используется классификация);
- реквизитов заказа (тип, [условия оплаты](../invoicing/settings.md#условия-оплаты), плановая дата и др.).

### Фильтры по датам

Как правило, в отчёте доступны:

- интервал дат;
- группировки по дню/неделе/месяцу/году (если включены соответствующие варианты отчёта).

### Для чего использовать

Отчёт полезен для:

- контроля «что заказано» и «у кого заказано»;
- анализа плановых сроков поставки;
- подготовки сверок с поставщиками.
