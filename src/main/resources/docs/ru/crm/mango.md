---
title: "Интеграция с телефонией Mango Office"
---

Модуль интеграции с IP-телефонией Mango Office позволяет объединить вашу облачную АТС с системой для автоматизации обработки вызовов, ведения истории общения с клиентами и прослушивания записей разговоров.

## 1. Подготовка в личном кабинете Mango Office

Прежде чем приступать к настройкам в системе, необходимо выполнить подготовительные действия в личном кабинете Mango Office:

1.  **Получение ключей доступа**:
    *   Перейдите в раздел управления интеграциями.
    *   Скопируйте значения **Уникальный код вашей АТС** и **Ключ для создания подписи**.
2.  **Настройка уведомлений**:
    *   В настройках программного интерфейса найдите раздел для указания адреса внешней системы.
    *   Укажите адрес в формате: `https://<адрес_вашей_системы>/exec/mango`.
    *   Выберите типы событий, о которых должна уведомлять АТС (Звонки, Записи).
3.  **Особенности настройки голосового меню**:
    *   Для работы автоматического распределения вызовов рекомендуется настроить проигрывание мелодии приветствия. Это даст системе время на идентификацию звонящего и отправку команды на перенаправление вызова ответственному сотруднику.

## 2. Настройка подключения в системе

Параметры подключения задаются в разделе настроек интеграций на вкладке **Mango**.

Необходимо заполнить следующие поля:
*   **Адрес** — адрес сервера телефонии (по умолчанию `https://app.mango-office.ru`).
*   **Код** — «Уникальный код вашей АТС», полученный в личном кабинете.
*   **Секрет** — «Ключ для создания подписи», полученный в личном кабинете.

## 3. Синхронизация и сопоставление

Для корректной работы системы необходимо загрузить данные о внутренних номерах и связать их с сотрудниками.

1.  На вкладке **АТС** нажмите кнопку **Загрузить пользователей АТС**.
2.  Система получит из облачной АТС список всех настроенных внутренних номеров.
3.  Для каждой загруженной строки в колонке **Сотрудник** выберите соответствующего человека из справочника системы. Это позволит закреплять историю звонков за конкретными людьми.

## 4. Обработка звонков

### Входящие звонки
При поступлении вызова система выполняет ряд автоматических действий:
*   **Определение клиента**: По номеру телефона система ищет связанные записи (например, открытые сделки или карточки партнеров).
*   **Автоматическая переадресация**: Если звонящий закреплен за определенным менеджером, система отправит команду в АТС на перевод вызова на внутренний номер этого сотрудника. Это происходит в тот момент, когда клиент слышит приветствие.
*   **Регистрация**: В разделе **Звонки** автоматически создается запись, в которой фиксируется время, участники и результат разговора.

### Исходящие звонки
Вы можете совершать вызовы одним нажатием на номер телефона в любой карточке системы:
1.  Нажмите на иконку телефона рядом с номером.
2.  Система инициирует **Обратный звонок**.
3.  Сначала зазвонит ваш телефон (или программа для звонков на компьютере).
4.  Как только вы поднимете трубку, начнется набор номера клиента.

## 5. Записи разговоров

Если ваша АТС настроена на запись разговоров, их можно прослушивать непосредственно в системе:

1.  В списке звонков для записей, которые уже обработаны АТС, доступна функция прослушивания.
2.  Нажмите кнопку **Запросить запись**.
3.  Система загрузит аудиофайл и откроет проигрыватель.
4.  Загруженные записи сохраняются в системе для быстрого повторного доступа.
