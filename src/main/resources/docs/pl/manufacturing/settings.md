## Lokalizacja

Otwórz **„Produkcja”** → **„Ustawienia”** → **„Ustawienia”**.

## Statusy zamówień produkcji

Ustawienia zawierają katalog statusów zamówień produkcji.

Co jest ważne:

- flaga **„Tylko do odczytu”** powoduje, że zamówienie nie jest edytowalne w danym statusie;
- statusy wpływają na to, jakie akcje są dostępne (rezerwacja, start, zakończenie).

## Typy zamówień produkcji

Ustawienia zawierają katalog typów zamówień produkcji. Zwykle typ definiuje:

- nazwę i identyfikator;
- numeratory (generowanie numeru);
- lokalizację materiałów;
- flagę demontażu;
- **typ Odpadu**.

Jeśli istnieje tylko jeden typ, jest używany jako domyślny.

## Rekomendowana kolejność konfiguracji

1. Utwórz typy zamówień produkcji.
2. Skonfiguruj numeratory i lokalizacje materiałów.
3. Skonfiguruj statusy i uprawnienia do edycji.
4. Jeśli trzeba, skonfiguruj **typ Odpadu**.
