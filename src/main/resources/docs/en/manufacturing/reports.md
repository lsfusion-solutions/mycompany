The reporting section is used to analyze [manufacturing orders](orders.md). Open **"Manufacturing" → "Reporting" → "Order report"**.

## What the report shows

The report is a pivot view with the following tabs:

- **Manufacturing orders** — one row per order: planned vs actual output (`to manufacture` / `manufactured`), order status, and the cost columns (material **cost**, **additional costs**, **labor costs**, **service costs**, **total cost**);
- **Materials** — one row per material line: planned vs actual consumption (`to consume` / `consumed`) and reserved quantity;
- **Labor cost** — one row per time entry: employee, project, worked hours, rate and labor amount. This tab is available when the Project Management contour is used.

The **Manufacturing orders** and **Materials** tabs can be grouped by item, item category, period and status.

## What is usually analyzed

- number of orders for a period;
- planned vs actual output;
- material availability and consumption;
- cost of produced goods;
- **Done** and **Canceled** orders.

## Usage recommendations

1. Set a date range.
2. Group by item and responsible person to see workload.
3. Compare planned and actual quantities to analyze deviations.
